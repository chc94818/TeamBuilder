🎮 **2D Team Lineup Editor**  
A React + TypeScript tool for building and previewing team rosters.

🖼️ **ASSETS**
- Put the background.jpg to the src/assets/groups/[group_name]/background/
- Put the [players_name].jpg to the  src/assets/groups/[group_name]/players/

🚀 **Features**
- **Dynamic Lineup**: Auto-creates slots from player assets; shows empty placeholders.
- **Grid Control**: Real-time adjustment of columns, gaps, and scale.
- **Smart Bench**: Click to auto-fill slots; prevents duplicate selection.
- **Responsive Cards**: Maintains correct aspect ratios.
- **UX Enhancements**: Right-click to remove, popup focus control, smooth scrolling.
- **Export**: Save lineup as high-resolution PNG with live layout tuning.

🛠 **Tech Stack**
- React (Hooks), TypeScript  
- Context-based state management  
- CSS Grid, Flexbox, react-rnd  

📖 **Usage**
- Select player → auto-fill lineup  
- Right-click to remove  
- Adjust layout via toolbar  
- Export as image
